SnapCart is a web-based Online Grocery Store application developed using the Django framework.  
The project provides separate roles for **Admin** and **Users** to manage products, orders, and customer interactions efficiently.

This project demonstrates full-stack web development concepts such as authentication, CRUD operations, cart management, and database integration using Django.

##  Features

- User Registration & Login
- Admin Login & Dashboard
- Product Management (Add / Update / Delete)
- Category & Subcategory Management
- Search Products (by name, category, price)
- Add to Cart & Remove from Cart
- Booking / Order Management
- Payment Option 
- View Orders & Booking History
- Change Password
- User Feedback System
- Contact Us Page
- Secure Login & Logout

##  User Roles

###  User
- Register & Login
- View Products
- Search Products
- Add to Cart
- Place Orders
- View Booking History
- Update Profile
- Send Feedback

###  Admin

- Login with Admin Credentials
- Add / Update / Delete Products
- Manage Categories
- View & Manage Users
- View Orders / Bookings
- Update Order Status
- View Feedback
